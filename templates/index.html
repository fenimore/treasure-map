About

The main use of this application is to compare craigslist free stuff postings by location. The distance of a free thing is directly related to its value. There are many treasures to be found, using this application, hopefully, you will find something near you worth giving a new home.

The Treasure Map requires the precise name of the stuff's city, without spaces, in order for the url to work -- it must conform to Craigslist's URL. New York is then newyork. Many cities should work 'out of the box' (likely those with one word names). Currently a handful of cities have been tested, like Montreal, Toronto, Philly and New York. Try: D.C. or somewhere random.
To try a city not listed on the navigation menu, simply enter the city name after the url (treasure.plyp.org).

Click here for the website source code or here for the freestuff-bot script. This treasure map is written in Python 3 using the Flask framework.
