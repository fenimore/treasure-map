About

The main use of this website is to compare craigslist free stuff postings by location. The distance of free stuff is directly related to its value.

The Treasure Map requires the precise name of the stuff's city, without spaces, in order for the url to work. New York is then newyork. This distrupts the coordinate getters and setters, but many cities should work (likely those with one word names). Currently a handful of cities have been tested, like Montreal, Toronto, Philly and New York. Try: D.C. or somewhere random.

Click here for the website source code or here for the freestuff-bot standalone. The application is written in Python3 using the Flask framework among other packages.
